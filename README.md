
Full Stack Store:

A full-stack e-commerce web application built using HTML, CSS, JavaScript, PHP, and MySQL.
The project simulates an online store with product listing, cart functionality, and checkout system using API-based communication between frontend and backend.
---------------------------------------------------------
 Features:
Display products dynamically from database
Add products to cart
Update cart quantities
Checkout system
REST-like API using PHP
Clean and responsive UI
----------------------------------------------------------
 Technologies Used:
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Local Server: XAMPP
----------------------------------------------------------
 Project Structure:
/assets → CSS, JavaScript, Images
/APIs → PHP backend files
/database → SQL database file
front.html → Main page
----------------------------------------------------------
 Setup Instructions:
Clone the repository
Import the database file from /database into phpMyAdmin
Run the project using XAMPP
Open front.html in your browser
----------------------------------------------------------
 Notes:
This project runs locally using XAMPP
Update API URLs when deploying to a live server
